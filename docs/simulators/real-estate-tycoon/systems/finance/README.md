# 资金系统

放资金相关代码：

- 银行开发贷、监管户回款、展期、抵押授信。
- 朋友借款、朋友入股、小贷、高息短钱。
- 债务、付息、到期压力和资金盘滚动。
- 资金页先选渠道，再选具体出钱人或窗口；右侧显示额度、月息和可借金额，不再展示抽象材料清单。
- 借款动作必须写清本金、月息、每轮利息和后果；银行、朋友、小贷、高利贷分别进入不同债务桶，付息要和选择匹配。
- 项目回款拆成签约、到账、自由现金和监管户；已售很快但工程接近交付时，按揭/监管释放要形成明显现金流，而不是每轮只回一两点。

约束：资金入口第一层应该是通用融资渠道；具体到某块土地或某个项目时，再进入抵押、还款、监管户等细项。
