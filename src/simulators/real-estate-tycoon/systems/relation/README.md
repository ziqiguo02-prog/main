# 关系系统

放关系相关代码：

- 关系分类：朋友、政府部门、银行、其他关系。
- 联系人列表和联系人动作。
- 借钱报价：额度、利息、对方回复、关系门槛。
- 共同开发/共同竞标：比例、话语权、退出条件。

约束：

- 政府联系人不直接出资，只提供审批、白名单、监管协调和被刁难时的保护；可以出现干股承诺，但这属于灰线利益输送，不是联合竞标出资。
- 联合投资类动作需要关系达到 80 以上。
- 点击联系人本身不能让关系变好；只有执行具体动作才改变关系。
- 土拍页问口径、选人、切分类都不能刷关系；亲密度增长放在本系统的走动、送礼、叙旧、帮忙和真实合作动作里，并且要写清现金或风险成本。
